# Connecting Your Wallet

## Supported Wallets

Nekomancer supports any EVM-compatible wallet:
- MetaMask
- OKX Wallet  
- Rabby
- Trust Wallet
- And more...

## Step-by-Step Guide

### 1. Go to Nekomancer
Visit [nekomancer.fun](https://www.nekomancer.fun)

### 2. Click "Connect Wallet"
You'll see a button in the top-right corner of the page.

### 3. Choose Your Wallet
Select your preferred wallet from the list. A popup will appear asking for permission.

### 4. Add Monad Network (if needed)
If Monad isn't in your wallet yet, you'll be prompted to add it:

| Setting | Value |
|---------|-------|
| Network Name | Monad |
| RPC URL | https://rpc3.monad.xyz/ |
| Chain ID | 143 |
| Currency Symbol | MON |
| Block Explorer | https://monvision.io |

### 5. Approve Connection
Click "Connect" or "Approve" in your wallet popup.

### 6. Done!
Your wallet address will appear in the top-right. You're ready to swap!

## Troubleshooting

### "Network not supported"
Make sure you're connected to Monad Mainnet (Chain ID: 143). Switch networks in your wallet.

### Wallet not appearing
- Refresh the page
- Make sure your wallet extension is unlocked
- Try a different browser

### Transaction stuck
Monad transactions are usually instant. If stuck:
1. Check [monvision.io](https://monvision.io) for your transaction
2. Wait a few seconds and refresh
3. Try increasing gas if it's a complex swap
