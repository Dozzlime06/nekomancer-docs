# Connect Wallet

Nekomancer uses RainbowKit. Any EVM wallet works.

---

## Supported Wallets

- MetaMask
- Coinbase Wallet
- WalletConnect (mobile wallets)
- Rainbow
- Rabby

---

## Connecting

1. Click **Connect Wallet** in the top right
2. Select your wallet
3. Approve the connection in your wallet
4. You're connected

---

## Monad Network Setup

First time on Monad? Your wallet will prompt you to add the network.

If not, add it manually:

| Setting | Value |
|---------|-------|
| Network Name | Monad |
| Chain ID | 143 |
| RPC URL | https://rpc.monad.xyz |
| Currency Symbol | MON |
| Block Explorer | https://monadvision.com |

---

## Switching Networks

If you're on the wrong network:

1. Click the network name in your wallet
2. Select Monad
3. Done

The app will prompt you automatically if you're on the wrong chain.
